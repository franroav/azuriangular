This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.1.7.

## Download

Open your terminal and go to the folder where you want to download to project. Then write in the console 'git clone https://github.com/franroav/azuriangular'.

## Install Dependencies

After that project was downloaded, execute the following command on you terminal 'npm install' to install your dependencies. 

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
